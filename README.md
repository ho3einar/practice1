Three-Digit Odd Numbers with Repeated Digits (No Zero)
Description
This Python script prints all three-digit numbers between 100 and 999 that satisfy the following conditions:
The number is three digits long.
The number is odd.
The number does not contain zero.
At least two digits are the same.
How It Works
The program:
Loops through numbers from 100 to 999.
Converts each number to a string to check individual digits.
Checks:
The number does not contain 0.
The number is odd.
At least two digits are equal.
Prints the number if all conditions are met.
Requirements
Python 3.x
